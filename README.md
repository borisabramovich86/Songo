# Songo

Split user tracks by Spotify Audio Features and create a playlist for each feature:
- Acoustic  
- High Energy  
- Low Energy   
- Instrumental  
- Positive  
- Negative  

Spotify Audio features:  
https://developer.spotify.com/documentation/web-api/reference/get-audio-features


set ENV variables:  
SPOTIFY_ID  
SPOTIFY_SECRET  

Written in GO  
`go mod tidy`  
`go run main.go`
